# Multi-Functional-App
# Multifunctional NLP and Image Generation Tool

## Overview
This project is a multifunctional AI tool that integrates multiple NLP and image generation models using Hugging Face.

## Features
- Text Summarization
- Sentiment Analysis
- Chatbot
- Question Answering
- Next Word Prediction
- Story Generation
- Image Generation

## Technologies Used
- Python
- Hugging Face Transformers
- Diffusers
- Streamlit

## How to Run

1. Clone the repository
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app/streamlit_app.py

## Notes
- Image generation may take time on CPU
- Pretrained models are used from Hugging Face
